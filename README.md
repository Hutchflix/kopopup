# kopopup
Popup window for images, iframes or html.

How to use:

$('.item-to-popup').koPopup({
    type: "image",
    gallery: true
});

<div class="item-to-popup" img-url="/url/of/image/to/show.jpg">
  <div class="ko-popup-content">
    Description HTML for the image can be placed here.
  </div>
</div>
